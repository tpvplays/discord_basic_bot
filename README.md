# discord_basic_bot

Repositório de um bot que reaje automaticamente para cada mensagem de um usuário!

Para instalar:

pip install .

Para executar:

Altere a variável AUHTOR_ID para o ID do seu usuário;
Altere o token para o seu token de desenvolvedor do discord;
e rode o comando:
python bot.py
